### **Paso 2** 

For $w = 5\%$, using the unit weight of water in $\text{kN/m}^3$ which is $\gamma_w = 9.81 \text{ kN/m}^3$, Calculate the zero-air void unit weight, we have:

$$\gamma_{zav} = \frac{\gamma_w}{w + \frac{1}{G_s}} = \frac{9.81}{0.05 + \frac{1}{2.75}} = \boxed{23.716 \text{ kN/m}^3}$$

### **Paso 3**

For $w = 8\%$:

$$\gamma_{zav} = \frac{9.81}{0.08 + \frac{1}{2.75}} = \boxed{22.113 \text{ kN/m}^3}$$

For $w = 10\%$:

$$\gamma_{zav} = \frac{9.81}{0.10 + \frac{1}{2.75}} = \boxed{21.159 \text{ kN/m}^3}$$

### **Paso 4** 

For $w = 12\%$:

$$\gamma_{zav} = \frac{9.81}{0.12 + \frac{1}{2.75}} = \boxed{20.284 \text{ kN/m}^3}$$

For $w = 15\%$:

$$\gamma_{zav} = \frac{9.81}{0.15 + \frac{1}{2.75}} = \boxed{19.099 \text{ kN/m}^3}$$
